# 📝 To-Do List Web App

This is a simple front-end To-Do List web application built using **HTML**, **CSS**, and **Vanilla JavaScript**.

## Features
- Add new tasks
- Mark tasks as completed
- Remove tasks
- Instant updates (no page reload)

## Technologies Used
- HTML5
- CSS3
- JavaScript (ES6)
- Live Server (for testing locally)
